# Go-LDAP-Admin 功能扩展计划

按优先级与依赖顺序逐项实现，完成一项勾选一项。

---

## Phase 1：快速见效（先做）

- [x] **1. 用户列表导出** - 当前列表/筛选结果导出 Excel（前端生成或后端接口）
- [x] **2. 最后登录时间** - 用户表字段、登录时更新、列表/筛选展示
- [x] **3. LDAP 连接状态** - 后端探测接口、首页或顶栏展示
- [x] **4. 首页最近操作** - Dashboard 展示操作日志最近 N 条，可跳转
- [x] **5. 系统信息页** - 版本、运行时长、DB 信息，仅管理员
- [x] **6. 表格列设置** - 用户/部门列表列显示隐藏、列宽记忆（localStorage）
- [x] **7. 敏感操作二次确认** - 删除用户/批量删除/重置密码前二次确认（输入确认文案或密码）

---

## Phase 2：同步与审计

- [x] **8. 同步预览（Dry Run）** - 同步前请求预览：新增/更新/跳过数量，不落库
- [ ] **9. 同步历史/日志** - 记录每次同步结果，列表可查
- [ ] **10. 登录日志** - 记录每次登录（用户、时间、IP、成功/失败），菜单下可查
- [ ] **11. 用户详情页** - 用户基本信息、所属部门、最近操作、快捷操作
- [ ] **12. 全局搜索** - 顶栏搜索用户/部门，下拉结果跳转
- [ ] **13. 同步概览** - 首页卡片：各来源上次同步时间、待同步数量

---

## Phase 3：批量与策略

- [ ] **14. 用户批量导入** - Excel/CSV 上传，列映射，校验后批量创建
- [ ] **15. 会话管理** - 当前登录设备列表、强制下线（可选）
- [x] **16. 账户过期策略** - 过期日或 N 天未登录自动禁用，列表/同步校验
- [ ] **17. 同步进度** - 大批量同步时进度条、可取消

---

## Phase 4：集成与配置

- [x] **18. Webhook** - 用户/部门创建或同步后 HTTP 回调
- [ ] **19. API 密钥** - 第三方调用用 API Key，管理页
- [ ] **20. 配置导入/导出** - 目录与同步规则等导出/导入 JSON

---

**当前进度**：从第 1 项开始，按序实现。
